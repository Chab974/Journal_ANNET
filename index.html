Portail Citoyen

Une une numérique pour la vie du village : grands sujets, rubriques, lecture rapide et actualités chargées depuis un flux JSON versionné.

Mode GitHub Pages

L'affichage est public et statique. L'analyse d'image et la génération des contenus se feront en amont, puis les résultats seront publiés dans data/citizen-posts.json.

Flux éditorial

Image ou affiche source → traitement ici avec Codex → mise à jour du JSON canonique → publication sur GitHub → affichage sur GitHub Pages.

Mode publication

Le site n’analyse rien lui-même. Il affiche uniquement les contenus déjà préparés et versionnés dans /data/citizen-posts.json.
